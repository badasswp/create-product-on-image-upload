Create WooCommerce products automatically by uploading images. Get new products to your e-commerce store faster and more efficiently.

== Installation ==

1. Go to 'Plugins > Add New' on your WordPress admin dashboard.
2. Search for 'Create Product On Image Upload' plugin from the official WordPress plugin repository.
3. Click 'Install Now' and then 'Activate'.
4. To get started, upload any image of your choice. Make sure the image has a clear name since that name would be used for creating the new WooCommerce product.
5. Make sure you have WooCommerce installed so that you can see the new product that has just been created.
6. That's it! You're all set to start using the Create Product on Image Upload plugin.

== Description ==

Create WooCommerce products automatically by uploading images.

This plugin makes it easy to create WooCommerce products by simply uploading the featured image of the product you intend to create. It will proceed to use the name of the image as the product name.

Save time and get more work done using this plugin. You'll never have to create WooCommerce products manually again!
